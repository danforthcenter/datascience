## Research computing and storage

The primary computing and data storage resources that support scientific research are hosted on Amazon Web Services (AWS).
We also have an on-premise HTCondor cluster for running jobs locally. For more information, see the [AWS](aws.md) and
[Computing with HTCondor](htcondor.md) sections of the documentation. A Jupyter service is available for interactive computing
and data analysis, and can be accessed at
[https://jupyter.datasci.danforthcenter.org/](https://jupyter.datasci.danforthcenter.org/).
